# 🚁 Drone Flight Simulator

A Python-based 3D drone simulator built from scratch using physics and control systems.

---

## ✨ Features
- 3D rigid-body physics
- PID altitude stabilization
- velocity-based flight control
- real-time 3D visualization
- keyboard-controlled drone movement

---

## 🎮 Controls
- W / S → forward/back
- A / D → left/right
- Q / E → yaw
- Arrow Up / Down → throttle

---

## 🧠 Concepts Used
- PID Control Systems
- Newtonian Physics
- Rotation Matrices
- Drone Flight Dynamics

---

## 🛠 Tech Stack
- Python
- NumPy
- Matplotlib
- pynput
